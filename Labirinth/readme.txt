Anleitung:

1. Laden Sie die Labirinth.exe Datei herunter. 
2.Speichern Sie die Datei an einem Ort, an dem Sie sie wiederfinden.
3.Führen Sie die heruntergeladene Datei jetzt aus.
4.Es kann sein, dass Windows Ihnen eine Sicherheitswarung sendet. Da dieses Spiel 100% Virenfrei ist, brauchen Sie sich keine Sorgen zu machen.
  Klicken Sie desshalb auf "Erweitert" und dann auf "Trotzdem Ausführen". 
5.Das Ziel des Spiels ist es, den Punkt in die grüne Fläche am ende des Pfades zu führen. Je länger sie brauchen desto mehr Punkte verlieren Sie. 
  Aber achtung, wenn Sie die Seiten berühren werden Ihnen Punkte abgezogen. 
  Ebenfalls führt das verlassen des Pfades dazu, dass Sie Game Over gehen.
  Am ende sehen sie Ihren persönlichen HighScore.

Vielen Dank für das Herunterladen & Spielen unseres Spiels.
